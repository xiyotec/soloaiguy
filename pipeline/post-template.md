## Style notes

- **Length:** 800–1500 words. Long enough to rank, short enough to finish.
- **Voice:** First person, direct, no hedging. "I tried X, it worked" not "One might try X."
- **Numbers > adjectives:** "saved 80% on tokens" beats "saved a lot."
- **Code blocks:** Always include language hint for syntax highlighting (` ```bash `, ` ```python `).
- **Affiliate links:** Disclose with `*` footnote. Only link to things I actually use.
- **No fluff intros:** Skip "In today's fast-paced world..." Start with the problem.
